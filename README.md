### 🧱 Engineering approach

- **One service, one responsibility** — bounded contexts in their own repos,
  behind a thin gateway that centralizes authentication.
- **Contract-first** — uniform, versioned API contracts (REST · GraphQL · gRPC).
- **Reproducible delivery** — generated data-access code, per-service image
  tagging, one-command deploys.
- **Ship and verify** — small increments, confirmed against the running system.
